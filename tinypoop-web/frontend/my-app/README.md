# React ProducCard

โปรเจคนี้มีส่วยประกอบหลักๆ 3 Component คือ

1. **Button** -ใช้สำหรับการกำหนดค่าปุ่มและเงื่อนไขของปุ่มตามที่เรากำหนดได้และดึงไปใช้จากส่วนอื่นๆได้
2. **ProductCard** -ใช้สำหรับการอ่านข้อมูลของสินค้าที่กรอกผ่าน หน้า app.tsx แล้วนำมาตกแต่งใส่กรอบการ์ดภายใน component productCard 
3.**NavBar** -ใช้แสดงไว้บนสุดของเว้บไซต์ตลอดเวลาและใส่เงื่อนไขที่ตรวจสอบขนาดหน้าจอเมื่อหน้าจมีขนาดเล็กเท่าจอมือถือให้ซ่อนเมนูและโชว์ ไอค่อนเมนูแทน เพื่อให้ผู้ใช้กดที่ไอค่อยจึงจะแสดงเมนูเพิ่มเติมให้กด
